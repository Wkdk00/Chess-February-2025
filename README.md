# Chess Console Game

Это консольное приложение для игры в шахматы на C++, разработанное для Windows.

## Примечание

Исходный код находится во второй ветке репозитория.

## Особенности

- **Традиционная шахматная доска** 8x8
- **Полный набор фигур** с корректными правилами перемещения
- **Визуализация возможных ходов** (отмечаются символом '+')
- **Поочередная игра** за белых и черных
- **Русская локализация** интерфейса

## Управление

1. При запуске отображается начальная позиция фигур
2. Игроки поочередно делают ходы:
   - Сначала выбирают фигуру (вводят координаты Y и X)
   - Затем выбирают клетку для хода (вводят координаты Y и X)

## Системные требования

- Windows OS
- Поддержка кириллицы в консоли
- Компилятор C++ с поддержкой стандарта C++11 или новее

## Особенности реализации

- Используется полиморфизм для различных типов фигур
- Реализованы все правила перемещения шахматных фигур
- Визуализация с помощью псевдографики в консоли
